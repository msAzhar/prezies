Git Nedir?

Git bir sürüm kontrol sistemidir.
(Daha detaylı olarak sunum sonunda anlatılmıştır.)

GitHub Nedir?

Git sürüm takip sistemini kullanan projeleri depolayan bir web servistir.

GitLab Nedir?

wiki ve iş takibi özelliklerine sahip olan bir açık kaynak lisanslı olan web-tabanlı Git repoların yöneticisidir.

GitHub ve GitLab Farkları

GitHub paralı. GitLab ise ücretsiz.

  • GitHub sunucuları uzakta olan sunuculardır
  • GitLab ise kendi yerel sunucunuzu kurmanıza imkan sağlıyor
  • GitLab, sadece sürüm takip değil de, wiki, iş yönetimi gibi özelliklere sahiptir
